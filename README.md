This application requires [NodeJS](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com/) to run. We assume that you have installed these tools on your system.

## Running the application

Install the dependencies:

```console
yarn install
```

Run the application:

```console
yarn start
```

## Running the tests

Install the dependencies:

```console
yarn install
```

Run the tests:

```console
yarn test
```
